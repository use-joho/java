# この章を進めるにあたり
特に注釈がない限り「 > 」で始まる行はコマンドの入力を表します。

例）以下のケースの場合コマンドプロンプトへ`java Sample`と入力し、その結果として`12345`が表示されています。
```
> java Sample
12345
```

# ファイルの実行

次のコマンドでコンパイル及び実行可能なJavaファイルを作成してください。

```bash
javac Sample.java
javac Test
```

# 実行と標準出力
以下のような出力結果となるJavaファイルを作成してください。
```bash
> javac Sample.java
> java Test
あいうえお
かきくけこ
```

----

以下のような出力結果となるJavaファイルを作成してください。

```bash
> javac Sample.java
> java Sample
123456789
```

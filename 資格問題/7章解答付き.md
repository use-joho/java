`ソースファイル名： Sample.java`
```java

	 	return 0;
	 }
}
class ChildTest{
	// ここにコードを挿入
}
```
「ここにコードを挿入」の位置に挿入する有効なコードはどれですか。2つ選択してください。

1. `int runnnig(int x) { return 1};`
1. `public int runnnig(int x) { return 0};`
